概要

cms2:

  • 小規模なブログ等のCMS 機能で

    markdown 形式の入力対応しております。

  • Vue.js (Vue CLI 3) +firebae構成となります。

コード

  • 通常

https://github.com/kuc-arc-f/vue_2_cms2

  • 表示専用

https://github.com/kuc-arc-f/vue_2_cms2_pub

環境

  • Vue.js
  • Vue-CLI 3
  • firebase

インストール方法

Vue CLI 側:

  • Vue-CLI 3 インストール

  • 上記の、github コードを、コピー

  • npm install

  • firebase の設定

    src/main.js に、設定下さい。

  • ビルドする場合、

    npm run build

  • 開発、修正する場合

    npm run serve
